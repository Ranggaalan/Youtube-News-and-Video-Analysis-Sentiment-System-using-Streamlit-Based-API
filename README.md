## üìå Overview
SmartNews Analyzer is an AI-driven web tool built with Streamlit that empowers users to search, analyze, and extract insights from news articles and YouTube videos. Leveraging APIs from NewsAPI and Google Cloud Console (YouTube API), this system provides deep sentiment analysis, visualization, and summarization of news content.
![image](https://github.com/user-attachments/assets/ae616ad3-b96e-43f7-885f-a5504b6005ac)

## üî• Features
- **Smart News & Video Search:** Instantly fetch news articles and YouTube videos based on custom keywords.
- **Insightful Metadata Extraction:** Access headlines, sources, publication dates, and direct links.
- **Advanced Sentiment Analysis:** Classify news as Positive, Neutral, or Negative.
- **Interactive Visualizations:** 
  - Sentiment distribution charts
  - AI-powered WordCloud for keyword analysis
  - Highlighting top positive & negative news stories
- **AI-Powered Summarization:** Generate quick summaries of sentiment trends.
- **Exportable Reports:** Download results in CSV format.
- **User-Friendly Guide:** Step-by-step manual for seamless user experience.
